# Slideshow

A basic macOS app that displays images in a folder as a full-screen slideshow.

## Installation

Build using Xcode, or `xcodebuild` from the command line.

## Keyboard Shortcuts (In-App)

 Key              | Action
 -----------------|-------------
 ⌘+⏎              | Open folder
 space or → or ↓  | Next photo
 ⌫ or ← or ↑      | Previous photo
 ⎋                | Exit
